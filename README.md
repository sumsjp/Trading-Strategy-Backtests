<summary>1. Previous Day High and Low Day Trading Strategy</summary><br>

| **条件** | **规则** |
|---------|---------|
| **进场** | **昨日高点突破（Breakout）** |
| **入场时间** | **日内开盘 - 收盘之间，但不在最后一小时进场** |

| **止盈（TP）** | **5:1 R:R，但若未达成则收盘强制平仓** |
| **日内强制平仓** | **若 SL 或 TP 未达成，15:30 自动平仓** |
